上传下载

前提

  • 已设置好配置项：系统配置、认证配置、主机配置

说明

  • 以\（Windows）或/（Linux）结尾代表文件夹
  • 上传时，local可以是文件或文件夹，remote默认是文件夹
  • 下载时，remote可以是文件或文件夹，local默认是文件夹

上传

将本地节点上的内容上传至远端目标主机组的每个主机上

文件：

$ xsh.exe -mode copy -group alpha02 -direction upload -local D:\tmp\LICENSE -remote /tmp/xyz
[192.168.165.37    ] ---------------------------------------------------------
D:\tmp\LICENSE -> /tmp/xyz/LICENSE :OK
[192.168.170.109   ] ---------------------------------------------------------
D:\tmp\LICENSE -> /tmp/xyz/LICENSE :OK

文件夹：

$ xsh.exe -mode copy -group alpha02 -direction upload -local D:\tmp\ -remote /tmp/xyz1
[192.168.170.109   ] ---------------------------------------------------------
D:\tmp\LICENSE -> /tmp/xyz1/LICENSE :OK
D:\tmp\hello -> /tmp/xyz1/hello :OK
D:\tmp\hello\README.md -> /tmp/xyz1/hello/README.md :OK
[192.168.165.37    ] ---------------------------------------------------------
D:\tmp\LICENSE -> /tmp/xyz1/LICENSE :OK
D:\tmp\hello -> /tmp/xyz1/hello :OK
D:\tmp\hello\README.md -> /tmp/xyz1/hello/README.md :OK

检查结果：

$ xsh.exe -mode cmd -group alpha02 -cmd "find /tmp/xyz*"
[192.168.165.37    ] ---------------------------------------------------------
/tmp/xyz
/tmp/xyz/LICENSE
/tmp/xyz1
/tmp/xyz1/LICENSE
/tmp/xyz1/hello
/tmp/xyz1/hello/README.md

[192.168.170.109   ] ---------------------------------------------------------
/tmp/xyz
/tmp/xyz/LICENSE
/tmp/xyz1
/tmp/xyz1/hello
/tmp/xyz1/hello/README.md
/tmp/xyz1/LICENSE

下载

将远端目标主机组的每个主机上的内容下载至本地节点，以目标IP进行文件夹分割

文件：

$ xsh.exe -mode copy -group alpha02 -direction download -local D:\xyz -remote /tmp/xyz/LICENSE
[192.168.165.37    ] ---------------------------------------------------------
D:\xyz\192.168.165.37\LICENSE <- /tmp/xyz/LICENSE :OK
[192.168.170.109   ] ---------------------------------------------------------
D:\xyz\192.168.170.109\LICENSE <- /tmp/xyz/LICENSE :OK


$ tree /F d:\xyz
D:\XYZ
├─192.168.165.37
│      LICENSE
│
└─192.168.170.109
        LICENSE

文件夹：

$ xsh.exe -mode copy -group alpha02 -direction download -local D:\xyz1 -remote /tmp/xyz1/
[192.168.170.109   ] ---------------------------------------------------------
D:\xyz1\192.168.170.109\hello <- /tmp/xyz1/hello :OK
D:\xyz1\192.168.170.109\hello\README.md <- /tmp/xyz1/hello/README.md :OK
D:\xyz1\192.168.170.109\LICENSE <- /tmp/xyz1/LICENSE :OK
[192.168.165.37    ] ---------------------------------------------------------
D:\xyz1\192.168.165.37\LICENSE <- /tmp/xyz1/LICENSE :OK
D:\xyz1\192.168.165.37\hello <- /tmp/xyz1/hello :OK
D:\xyz1\192.168.165.37\hello\README.md <- /tmp/xyz1/hello/README.md :OK


$ tree /F d:\xyz1
D:\XYZ1
├─192.168.165.37
│  │  LICENSE
│  │
│  └─hello
│          README.md
│
└─192.168.170.109
    │  LICENSE
    │
    └─hello
            README.md
