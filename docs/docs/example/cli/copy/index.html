上传下载

前提

  • 已设置好配置项：系统配置、认证配置、主机配置

说明

  • 件或文件夹必须是全路径，例如：D:\tmp\LICENSE（Windows）或/tmp/abc（Linux）
  • 上传时，local可以是文件或文件夹，remote默认是文件夹
  • 下载时，remote可以是文件或文件夹，local默认是文件夹
  • 仅支持以默认用户执行，不支持切换用户执行

末尾带路径分隔符表示文件夹，否则表示文件

上传

将本地节点上的内容上传至远端目标主机组的每个主机上

文件：

$ xsh-windows-amd64.exe -mode copy -group alpha02 -direction upload -local D:\tmp\LICENSE -remote /tmp/xsh/copy3/file
[192.168.170.109   ] ---------------------------------------------------------
D:\tmp\LICENSE -> /tmp/xsh/copy3/file/LICENSE :FILE:OK
[192.168.165.37    ] ---------------------------------------------------------
D:\tmp\LICENSE -> /tmp/xsh/copy3/file/LICENSE :FILE:OK

文件夹：

$ xsh-windows-amd64.exe -mode copy -group alpha02 -direction upload -local D:\tmp\hello\ -remote /tmp/xsh/copy3/dir
[192.168.170.109   ] ---------------------------------------------------------
D:\tmp\hello\ -> /tmp/xsh/copy3/dir/hello :DIR:OK
[192.168.165.37    ] ---------------------------------------------------------
D:\tmp\hello\ -> /tmp/xsh/copy3/dir/hello :DIR:OK

下载

将远端目标主机组的每个主机上的内容下载至本地节点，按目标IP新建文件夹进行隔离

文件：

$ xsh-windows-amd64.exe -mode copy -group alpha02 -direction download -local D:\tmp\copy3 -remote /tmp/xsh/copy3/file/LICENSE
[192.168.170.109   ] ---------------------------------------------------------
D:\tmp\copy3\192.168.170.109\LICENSE <- /tmp/xsh/copy3/file/LICENSE :FILE:OK
[192.168.165.37    ] ---------------------------------------------------------
D:\tmp\copy3\192.168.165.37\LICENSE <- /tmp/xsh/copy3/file/LICENSE :FILE:OK

文件夹：

$ xsh-windows-amd64.exe -mode copy -group alpha02 -direction download -local D:\tmp\copy5 -remote /tmp/xsh/copy3/dir/
[192.168.170.109   ] ---------------------------------------------------------
D:\tmp\copy5\192.168.170.109\dir <- /tmp/xsh/copy3/dir/ :DIR:OK
[192.168.165.37    ] ---------------------------------------------------------
D:\tmp\copy5\192.168.165.37\dir <- /tmp/xsh/copy3/dir/ :DIR:OK
