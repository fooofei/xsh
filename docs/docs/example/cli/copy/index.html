上传下载

前提

  • 已设置好配置项：系统配置、认证配置、主机配置

说明

  • 上传时，local可以是文件或文件夹，remote默认是文件夹
  • 下载时，remote可以是文件或文件夹，local默认是文件夹
  • 仅支持以默认用户执行，不支持切换用户执行

上传

将本地节点上的内容上传至远端目标主机组的每个主机上

文件：

$ xsh-windows-amd64.exe -mode copy -group alpha02 -direction upload -local D:\tmp\LICENSE -remote /home/abc/cli/copy
[192.168.170.109   ] ---------------------------------------------------------
D:\tmp\LICENSE -> /home/abc/cli/copy/LICENSE :OK
[192.168.165.37    ] ---------------------------------------------------------
D:\tmp\LICENSE -> /home/abc/cli/copy/LICENSE :OK

文件夹：

$ xsh-windows-amd64.exe -mode copy -group alpha02 -direction upload -local D:\tmp -remote /home/abc/cli/copy/folder
[192.168.165.37    ] ---------------------------------------------------------
D:\tmp -> /home/abc/cli/copy/folder/tmp :OK
D:\tmp\LICENSE -> /home/abc/cli/copy/folder/tmp/LICENSE :OK
D:\tmp\hello -> /home/abc/cli/copy/folder/tmp/hello :OK
D:\tmp\hello\README.md -> /home/abc/cli/copy/folder/tmp/hello/README.md :OK
[192.168.170.109   ] ---------------------------------------------------------
D:\tmp -> /home/abc/cli/copy/folder/tmp :OK
D:\tmp\LICENSE -> /home/abc/cli/copy/folder/tmp/LICENSE :OK
D:\tmp\hello -> /home/abc/cli/copy/folder/tmp/hello :OK
D:\tmp\hello\README.md -> /home/abc/cli/copy/folder/tmp/hello/README.md :OK

下载

将远端目标主机组的每个主机上的内容下载至本地节点，按目标IP新建文件夹进行隔离

文件：

$ xsh-windows-amd64.exe -mode copy -group alpha02 -direction download -local D:\tmp\cli -remote /home/abc/cli/copy/LICENSE
[192.168.170.109   ] ---------------------------------------------------------
D:\tmp\cli\192.168.170.109\LICENSE <- /home/abc/cli/copy/LICENSE :OK
[192.168.165.37    ] ---------------------------------------------------------
D:\tmp\cli\192.168.165.37\LICENSE <- /home/abc/cli/copy/LICENSE :OK

$ xsh-windows-amd64.exe -mode copy -group alpha02 -direction download -local D:\tmp\cli\folder -remote /home/abc/cli/copy
