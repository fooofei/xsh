任务编排

前提

  • 已设置好配置项：系统配置、认证配置、主机配置

说明

example\show-info.yaml

name: show info
actions:
  - name: show mem and disk
    group: alpha02
    steps:
      - type: command
        commands:
          - free -h
          - df -h
  - name: show user info
    group: alpha03
    steps:
      - type: command
        commands:
          - id
  • 在alpha02上查看内存和磁盘信息
  • 在alpha03上查看用户信息
