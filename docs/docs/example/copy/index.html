上传现在

前提

  • 已设置好配置项：系统配置、认证配置、主机配置
  • 已切换到目标主机组环境
  • 已切换至目标动作环境

以group=alpha02为例

上传

将本地节点上的内容上传至远端目标主机组的每个主机上

用法：local -> remote

约束：

  • local和remote必须是全路径，例如：D:\tmp\LICENSE（Windows）或/tmp/abc（Linux）
  • local可以是文件或文件夹，以\（Windows）或/（Linux）结尾代表文件夹
  • remote必须是文件夹，如果不存在将自动在远端新建文件夹

上传文件

[alpha02:copy]# D:\tmp\LICENSE -> /tmp/abc
..
[192.168.165.37    ] ---------------------------------------------------------
D:\tmp\LICENSE -> /tmp/abc/LICENSE :OK
[192.168.170.109   ] ---------------------------------------------------------
D:\tmp\LICENSE -> /tmp/abc/LICENSE :OK

检查结果（任一目标节点）：

# find /tmp/abc
/tmp/abc
/tmp/abc/LICENSE

上传文件夹

[alpha02:copy]# D:\tmp\ -> /tmp/abc1
..
[192.168.170.109   ] ---------------------------------------------------------
D:\tmp\LICENSE -> /tmp/abc1/LICENSE :OK
D:\tmp\hello -> /tmp/abc1/hello :OK
D:\tmp\hello\README.md -> /tmp/abc1/hello/README.md :OK
[192.168.165.37    ] ---------------------------------------------------------
D:\tmp\LICENSE -> /tmp/abc1/LICENSE :OK
D:\tmp\hello -> /tmp/abc1/hello :OK
D:\tmp\hello\README.md -> /tmp/abc1/hello/README.md :OK

检查结果（任一远端目标节点）：

# find /tmp/abc1
/tmp/abc1
/tmp/abc1/hello
/tmp/abc1/hello/README.md
/tmp/abc1/LICENSE

下载

将远端目标主机组的每个主机上的内容下载至本地节点，以目标IP进行文件夹分割

用法：local <- remote

约束：

  • local和remote必须是全路径，例如：D:\tmp\LICENSE（Windows）或/tmp/abc（Linux）
  • remote可以是文件或文件夹，以\（Windows）或/（Linux）结尾代表文件夹
  • local必须是文件夹，如果不存在将自动在本地新建文件夹

下载文件

[alpha02:copy]# D:\hello <- /tmp/abc1/LICENSE
..
[192.168.170.109   ] ---------------------------------------------------------
D:\hello\192.168.170.109\LICENSE <- /tmp/abc1/LICENSE :OK
[192.168.165.37    ] ---------------------------------------------------------
D:\hello\192.168.165.37\LICENSE <- /tmp/abc1/LICENSE :OK

检查结果（本地节点）：

$ tree /F D:\hello
D:\HELLO
├─192.168.165.37
│      LICENSE
│
└─192.168.170.109
        LICENSE

下载文件夹

[alpha02:copy]# D:\hello1 <- /tmp/abc1/
..
[192.168.165.37    ] ---------------------------------------------------------
D:\hello1\192.168.165.37\LICENSE <- /tmp/abc1/LICENSE :OK
D:\hello1\192.168.165.37\hello <- /tmp/abc1/hello :OK
D:\hello1\192.168.165.37\hello\README.md <- /tmp/abc1/hello/README.md :OK
[192.168.170.109   ] ---------------------------------------------------------
D:\hello1\192.168.170.109\hello <- /tmp/abc1/hello :OK
D:\hello1\192.168.170.109\hello\README.md <- /tmp/abc1/hello/README.md :OK
D:\hello1\192.168.170.109\LICENSE <- /tmp/abc1/LICENSE :OK

检查结果（本地节点）：

$ tree /F D:\hello1
D:\HELLO1
├─192.168.165.37
│  │  LICENSE
│  │
│  └─hello
│          README.md
│
└─192.168.170.109
    │  LICENSE
    │
    └─hello
            README.md
