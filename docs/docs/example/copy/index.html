上传下载

前提

  • 已设置好配置项：系统配置、认证配置、主机配置
  • 已切换到目标主机组环境

以group=alpha02为例

切换上下文

命令：:copy

[alpha02:do]# :copy
[alpha02:copy]#

上传

将本地节点上的内容上传至远端目标主机组的每个主机上

用法：local -> remote

约束：

  • local和remote必须是全路径，例如：D:\tmp\LICENSE（Windows）或/tmp/abc（Linux）
  • local可以是文件或文件夹
  • remote必须是文件夹，如果不存在将自动在远端新建文件夹

文件：

[alpha02:copy]# D:\tmp\LICENSE -> /home/abc/copy/tmp
[192.168.165.37    ] ---------------------------------------------------------
D:\tmp\LICENSE -> /home/abc/copy/tmp/LICENSE :OK
[192.168.170.109   ] ---------------------------------------------------------
D:\tmp\LICENSE -> /home/abc/copy/tmp/LICENSE :OK

[alpha02:copy]# :do
[alpha02:do]# find /home/abc/copy/tmp
[192.168.170.109   ] ---------------------------------------------------------
/home/abc/copy/tmp
/home/abc/copy/tmp/LICENSE

[192.168.165.37    ] ---------------------------------------------------------
/home/abc/copy/tmp
/home/abc/copy/tmp/LICENSE

[alpha02:do]# :copy
[alpha02:copy]#

文件夹：

[alpha02:copy]# D:\tmp -> /home/abc/copy/folder
[192.168.170.109   ] ---------------------------------------------------------
D:\tmp -> /home/abc/copy/folder/tmp :OK
D:\tmp\LICENSE -> /home/abc/copy/folder/tmp/LICENSE :OK
D:\tmp\hello -> /home/abc/copy/folder/tmp/hello :OK
D:\tmp\hello\README.md -> /home/abc/copy/folder/tmp/hello/README.md :OK
[192.168.165.37    ] ---------------------------------------------------------
D:\tmp -> /home/abc/copy/folder/tmp :OK
D:\tmp\LICENSE -> /home/abc/copy/folder/tmp/LICENSE :OK
D:\tmp\hello -> /home/abc/copy/folder/tmp/hello :OK
D:\tmp\hello\README.md -> /home/abc/copy/folder/tmp/hello/README.md :OK

[alpha02:copy]# :do
[alpha02:do]# find /home/abc/copy/folder
[192.168.170.109   ] ---------------------------------------------------------
/home/abc/copy/folder
/home/abc/copy/folder/tmp
/home/abc/copy/folder/tmp/hello
/home/abc/copy/folder/tmp/hello/README.md
/home/abc/copy/folder/tmp/LICENSE

[192.168.165.37    ] ---------------------------------------------------------
/home/abc/copy/folder
/home/abc/copy/folder/tmp
/home/abc/copy/folder/tmp/hello
/home/abc/copy/folder/tmp/hello/README.md
/home/abc/copy/folder/tmp/LICENSE

[alpha02:do]# :copy
[alpha02:copy]#

下载

将远端目标主机组的每个主机上的内容下载至本地节点，按目标IP新建文件夹进行隔离

用法：local <- remote

约束：

  • local和remote必须是全路径，例如：D:\tmp\LICENSE（Windows）或/tmp/abc（Linux）
  • remote可以是文件或文件夹
  • local必须是文件夹，如果不存在将自动在本地新建文件夹

文件：

[alpha02:copy]# D:\tmp\copy <- /home/abc/copy/tmp/LICENSE
[192.168.165.37    ] ---------------------------------------------------------
D:\tmp\copy\192.168.165.37\LICENSE <- /home/abc/copy/tmp/LICENSE :OK
[192.168.170.109   ] ---------------------------------------------------------
D:\tmp\copy\192.168.170.109\LICENSE <- /home/abc/copy/tmp/LICENSE :OK

$ tree /F D:\tmp\copy
D:\TMP\COPY
├─192.168.165.37
│      LICENSE
│
└─192.168.170.109
        LICENSE
