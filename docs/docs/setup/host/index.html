主机组配置

.xsh/host.yaml

hosts:
  - name: alpha-123
    auth: alpha
    port: 22
    addresses:
      - 192.168.187.117
    groups:
      - alpha01
      - alpha02
    details:
      - address: 192.168.184.236
        port: 22
        username: abc
        password: XXXXXX
        sutype: "sudo -S su -"
        supass: XXXXXX
        privatekey: ""
        passphrase: ""

无默认值，hosts为列表类型

配置说明

  - name: alpha-123                       # 名称
    auth: alpha                           # 认证名称，需要和auth.yaml中名称配置项一致
    port: 22                              # 端口，可选，默认：22
    addresses:                            # 地址列表
      - 192.168.187.117
    groups:                               # 主机组列表，可选，名称需要和host.yaml中名称配置项一致
      - alpha01
      - alpha02
    details:                              # 特殊配置项的主机列表信息，可选，配置项和auth含义保持一致
      - address: 192.168.184.236          # 地址
        port: 22                          # 端口，可选，默认：22
        username: abc
        password: XXXXXX
        sutype: "sudo -S su -"
        supass: XXXXXX
        privatekey: ""
        passphrase: ""

addresses地址列表另外支持IPv4地址段配置，例如192.168.187.120-192.168.187.150，包含两端的IP地址，仅支持最后16位变化。

addresses、groups和details，三种类型配置，至少提供一种即可
