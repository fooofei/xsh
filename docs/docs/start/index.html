快速开始

构建

git clone https://github.com/xied5531/xsh.git
cd xsh
go build

帮助

$ xsh.exe -h

准备

  • 配置认证信息
  • 配置主机组信息

开始

  • 临时在主机组上执行命令
  • 临时往主机组上传输文件
  • 通过任务脚本化在主机组上执行组合动作
  • 需要常驻在主机组上长时间执行命令或传输文件
